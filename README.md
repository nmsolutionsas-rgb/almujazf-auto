# Al Mujazf Auto — Website

Cinematic marketing website for **Al Mujazf Auto**, a luxury & armored vehicle dealership on Salim Street, Sulaymaniyah, Iraq (est. 2009).

## Features

- Scroll-driven cinematic intro: exterior facade → armored Brabus G-Class guards with live siren glow → showroom aisle → gold brand reveal
- Curated collection grid (Revuelto, Huracán Sterrato, Urus, Brabus G800, Cullinan, Maybach S680) with WhatsApp enquiry links
- Trilingual: English, Arabic and Kurdish (Sorani) with full RTL support
- Contact section with phones, email, Instagram, opening hours and Google Maps link
- All imagery AI-generated for presentation purposes

## Stack

- Vite + React 19 + TypeScript
- Tailwind CSS v4
- GSAP ScrollTrigger

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build

```
